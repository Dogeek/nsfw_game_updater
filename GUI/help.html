Help for NSFW Game Manager

This is the help for NSFW Game Manager
This project allows people to keep track of NSFW games they enjoy. The latest GUI release is on github
To be able to edit the database, you'll need to fill out this form.
You can also open a pull request on github if you make database changes, we'll review it, and merge it if no errors have been found.

How to Use

Keybindings

CTRL+C : allows you to copy the game info to your clipboard
CTRL+D : copies the download link for your platform, according to the platform combobox, at the bottom of the screen.
CTRL+W : Mark a game as downloaded, to check if the game has updates
CTRL+Q : Open the selected game's patreon page
HOME, END, PGUP, PGDOWN : these do what you'd expect them to

Contextual Menu

You can access the contextual menu by right clicking on a game in the list.

From then, you can access any function availaible through keybindings, plus the possibility to hide games, go to the graphtreon page and edit the entry in the database.

The search feature

To use the search feature, you must first input the name of the column you wish to search in, then a semicolon (":") and the search term afterwards.
For instance, if I want to search for games made by darkcookie, I'll enter :
   developer:darkcookie

If you want to filter multiple columns at once, you can. Separate every search term with a comma (",")
Example :
   visual_style:3D,animation:yes

Below is a full list of supported keys for the search engine :

  • animation
  • developer
  • download_link_android
  • download_link_linux
  • download_link_mac
  • download_link_windows
  • engine
  • game
  • genre
  • graphtreon
  • public_build
  • setting
  • visual_style

Game Rating :

You may rate any game. Just right-click on the game you want to rate, and select the grade you want from the "Rate game" submenu.
You may rate each game only once.

Top-level windows description

Options Window

The options window allows you to resize and choose an installation/download path. It also allows you to setup your accounts for downloading the files besides the provided (default) ones
You may setup these accounts in the "Advanced" tab of the options window.

Adding a New Game

You may add a new game by clicking the "Add New Game" item in the File menu.
Fill in the fields, all of them are required. If a game doesn't support a particular operating system, fill in a dash ("-") instead of the link.
If some fields are left void, the game will not be saved to the database.

If a game has no link for a given platform, use a dash ("-") instead. If you feel like some genres, settings, or engines are missing, please open a github issue and we'll add it.

Editing an existing Game

You may edit existing games by going in the "Edit>Edit Selected Entry" menu. It will open a window similar to the "Add new game" window, you can edit from there and click "Ok" to save your changes

Features :

  • Downloading and Updating games from the following sources : mega.nz, patreon, dropbox, google drive, itch.io and via apache.
  • Editing games in the database
  • Copying the links to the clipboard
  • Copying the whole line's data to the clipboard
  • check for game updates
  • Sorting games by genre, developer, etc.
  • A super awesome help page (this!)
  • Options to set the window size, the download path etc
  • Syncing every user's databases
  • Automatic unzipping/install of the games to the specified installation directory
  • Scrape reddit r/lewdgames and r/nsfwgaming for updates, and new releases to add them to the database
  • A search engine to sort through the games faster !

Planned :

  • Patreon integration to pledge from the app itself
